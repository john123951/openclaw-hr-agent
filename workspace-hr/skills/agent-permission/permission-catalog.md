# OpenClaw 权限速查清单

> 本文档是 HR 在审批权限申请时的参考手册。
> 权限数量不固定——内置工具固定，但已安装的 plugin 会动态扩展。
> 如需查询当前系统实际可用的全部权限，请运行发现脚本（见文末）。

---

## 🔒 基线权限（所有员工必须拥有）

| 权限 | 用途 | 为什么必须有 |
|------|------|------------|
| `sessions_list` | 查看公司同事列表 | 找到 HR/IT 的联系方式 |
| `sessions_send` | 给同事发消息 | 向 HR 申请权限、向 IT 报故障 |
| `sessions_history` | 查看会话历史 | 跟踪上下文、避免重复沟通 |
| `memory_search` | 搜索记忆库 | 检索历史经验和知识 |
| `memory_get` | 获取记忆内容 | 读取已保存的记忆条目 |
| `message` | 消息工具 | 向绑定渠道发送消息 |

---

## 📦 内置工具分组

### group:fs — 文件系统

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `read` | 读取文件 | 🟢 低 | **所有（基线）** | 必须有 |
| `write` | 写入/创建文件 | 🟢 低 | **所有（基线）** | 知识沉淀必须 |
| `edit` | 编辑现有文件 | 🟢 低 | **所有（基线）** | 维护记忆必须 |
| `apply_patch` | 应用补丁 | 🟡 中 | 开发 | 用于代码修改 |

### group:runtime — 运行时

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `exec` | 执行 shell 命令 | 🟠 中高 | 大多数岗位 | 是运行脚本和调用工具的基础 |
| `process` | 进程管理 | 🟠 中高 | 开发、运维 | 管理后台进程 |

### group:sessions — 会话协作

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `sessions_list` | 列出会话/同事 | 🟢 低 | **所有（基线）** | 生命线权限 |
| `sessions_send` | 发送消息给同事 | 🟢 低 | **所有（基线）** | 生命线权限 |
| `sessions_history` | 查看会话历史 | 🟢 低 | **所有（基线）** | 生命线权限 |
| `sessions_spawn` | 创建子 Agent | 🔴 高 | CEO 顾问、高级管理 | 可产生新的运行实例，需老板确认 |
| `session_status` | 查看会话状态 | 🟢 低 | 需要时授予 | 辅助诊断 |

### group:memory — 记忆

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `memory_search` | 搜索记忆库 | 🟢 低 | **所有（基线）** | 生命线权限 |
| `memory_get` | 获取记忆条目 | 🟢 低 | **所有（基线）** | 生命线权限 |

### group:web — 网络访问

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `web_fetch` | 抓取网页内容 | 🟢 低 | **所有（基线）** | 上网查资料的基本能力 |
| `web_search` | 网络搜索 | 🟡 中 | 调研、运营 | 需要 Brave API Key |

### group:ui — 界面工具

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `browser` | 浏览器操作 | 🟠 中高 | 调研、监控、测试 | 可访问网页，注意安全 |
| `canvas` | 画布/可视化 | 🟡 中 | 极少使用 | 一般禁止 |

### group:messaging — 消息

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `message` | 消息推送 | 🟢 低 | **所有（基线）** | 对外通讯的渠道 |

### group:nodes — 节点

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `nodes` | 远程节点管理 | 🔴 高 | 仅管理员 | 一般禁止 |

### 其他

| 权限 | 功能 | 风险等级 | 适用岗位 | 注意事项 |
|------|------|---------|---------|---------|
| `image` | 图片生成 | 🟡 中 | 设计、文案 | 调用图片生成模型 |
| `agents_list` | 列出所有 Agent | 🟢 低 | 按需 | 查看系统中注册的 Agent |

## 🔌 已安装 Plugin 扩展的权限

Plugin 安装后可能注册额外的工具名称。这些权限不在上表中——请运行以下命令动态发现：

```bash
$HOME/.openclaw/workspace-hr/skills/agent-permission/scripts/hr-permission-discover.sh
```

## ⚡ HR 审批快速决策树

```
员工申请权限
    │
    ├─ 是基线权限？(write, edit, web_fetch, sessions_*, memory_*, message)
    │   └─ ✅ 直接批复（这些本就该有）
    │
    ├─ 在岗位模板 allow 中？
    │   └─ ✅ 直接批复（模板已推荐）
    │
    ├─ 风险等级 🟢/🟡？
    │   └─ ✅ HR 可自行批复（记录原因）
    │
    ├─ 风险等级 🟠？
    │   └─ ⚠️ HR 评估合理性后批复
    │
    └─ 风险等级 🔴？(sessions_spawn, nodes)
        └─ ❌ 需老板确认
```
