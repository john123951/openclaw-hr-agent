# 我的项目目录

这是你存放个人项目的地方。

## 使用说明

- 在这里创建你的项目文件夹
- 每个项目应该有独立的目录

## 项目结构示例

```
projects/
├── project-a/
│   ├── src/
│   ├── tests/
│   └── README.md
├── project-b/
│   ├── daily-notes/
│   ├── references/
│   └── README.md
└── project-c/
    ├── datasets/
    ├── models/
    └── README.md
```

## 注意事项

- 不要在此目录存放敏感信息（如密钥、凭证）
- 大型二进制文件请使用 .gitignore 排除
